## File Formats

All packs include:
- **High-res WAV files** (24-bit/96kHz) - Professional quality
- **MP3 previews** (320kbps) - Quick audition
- **Metadata** - Detailed recording information
- **License** - Clear usage terms

## How to Use

1. Click on any sound pack to view details
2. Preview audio files using the built-in player
3. Download individual files or complete packs
4. Check the README in each pack for licensing info

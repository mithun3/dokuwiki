## âš¡ Quick Start Checklist

### Before You Record
- [ ] Scout your location for optimal recording conditions
- [ ] Check weather and ambient noise levels
- [ ] Test all equipment and batteries beforehand
- [ ] Plan your microphone setup and placement

### During Recording
- [ ] Always use headphones for real-time monitoring
- [ ] Record at **24-bit / 48kHz minimum** (or 96kHz for flexibility)
- [ ] Capture **room tone** (10-30 seconds of ambient silence)
- [ ] Take multiple takes from different angles
- [ ] Document everything (location, gear, settings, time)

### After Recording
- [ ] Label files with date, location, and content
- [ ] Backup files immediately to multiple locations
- [ ] Create organized folder structure
- [ ] Add metadata and notes for future reference
